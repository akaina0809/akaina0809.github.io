What is the skin change addon?

This is a convenient addon that solves the problem without changing the texture of the skin.

The following languages ​​are supported.

ja_JP 日本語 (日本)
en_US English (United States)

However, please note that Minecraft's default skin (other than skin packs) is Steve. And this site is English.

The skin change add-on uses beta (script) using

player.json It doesn't work well with other addons._________________
@minecraft/server Easily corrupted by Minecraft updates.__________________
@minecraft/server-ui Easily corrupted by Minecraft updates.__________________

There is a possibility that it will be broken (or unusable) depending on the Minecraft update.
