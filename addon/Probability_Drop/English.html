What is the Probability Drop Addon?

An addon that recreates the "5% Drop" project from Nichijou-gumi.

The following languages are supported:

ja_JP 日本語 (日本)
en_US English (United States)
ko_KR 한국어 (대한민국))

Please note that this is not a complete recreation....

This site is in English.

This addon uses @minecraft/server and @minecraft/server-ui with a stable script implementation.

Therefore, it is less likely to break due to Minecraft updates.

If you find bugs or if the addon no longer works, please contact me via X (formerly Twitter) or Discord.

Features

1. Items (blocks only) drop with a 5% probability

2. Adds custom blocks and items (original content)

3. Uses the Script API (for Bedrock Edition, not Java Edition)

4. Players with the "op" tag can configure the addon

5. Settings are saved in the world and are shared across players
