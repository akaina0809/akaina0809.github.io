What is the Probability Drop Addon?

An addon that recreates the "5% Drop" project from Nichijou-gumi.

Supported languages:

ja_JP日本語 (日本)
en_USEnglish (United States)
ko_KR한국어 (대한민국)

Note: this is not a complete recreation.

This site is in English.

This addon uses @minecraft/server and @minecraft/server-ui with a stable script implementation.

It is less likely to break due to Minecraft updates.

If you find bugs or the addon no longer works, please contact me via X or Discord.

Features

  • 5% chance for items to drop (blocks only)
  • Custom blocks and items added (original content)
  • Uses Script API (Bedrock Edition, not Java Edition)
  • Players with "op" tag can configure the addon
  • Settings are saved in world and shared
