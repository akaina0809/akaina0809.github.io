Probability Drop Addonとは？

日常組の5%dropの企画の再現addon

対応言語：

ja_JP日本語 (日本)
en_USEnglish (United States)

注意点：完全再現ではありません。

このサイトは日本語(Japanese)です。

このaddonは @minecraft/server と @minecraft/server-ui を使用した安定板(script)を使用しています。

アップデートによって壊れる可能性は低めです。

バグ報告は X または Discordまで。

内容一覧

  • 5％の確率でアイテムdrop (ブロックのみ)
  • ブロックとアイテム追加 (オリジナル要素)
  • 統合版なのでScript APIを使用
  • opタグでaddon設定可能
  • 設定はworldに保存され共有
